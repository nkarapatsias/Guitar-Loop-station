This project explores the use of an Arduino UNO to build a custom guitar looper pedalboard, designed to work seamlessly with Mobius 3, a powerful looping plugin hosted inside Reaper DAW. The goal is to create a versatile, professional-grade looper with three core modes of operation.

Main Modes:

Record Mode – This mode enables users to record and overdub multiple layers across different tracks independently. It's ideal for capturing guitar parts, vocals, or other instruments in a flexible and creative way.

Play Mode – After recording, users can switch to Play Mode to control playback. Each track can be muted or unmuted individually, allowing for dynamic live performances or complex arrangements.

Stop Mode – Integrated into Play Mode, Stop Mode allows users to instantly mute all tracks and selectively unmute only the ones they wish to hear. It offers a quick and intuitive way to reset or shift the arrangement on the fly.

The looper also offers a range of customization options. Users can select which input(s) are routed to each track, and the system stays tightly synced with the host DAW’s metronome—ensuring everything stays rhythmically aligned.

In summary, this project demonstrates how affordable hardware like the Arduino UNO can be combined with professional software to create a capable and customizable looping solution for musicians.
